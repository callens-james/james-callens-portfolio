# LifeOps Copilot — Master Product Spec

## Product Goal
Provide a single, local-first system that transforms incoming life/work obligations into prioritized, trackable, and safely executable action plans.

## Target users
- Everyday households
- Job seekers / career changers
- Solo operators / small businesses
- Operations-focused teams

## Universal Core
1. Unified Inbox (files/forms/notes/tasks)
2. Extractor (deadlines, entities, required docs)
3. Priority + Risk Engine
4. Planner + workflow states
5. Trust Layer (approve/undo/audit)
6. Background agent (reminders, stale-task checks, weekly summaries)

## Integrated Modules (MVP level)
- Paperwork Autopilot
- Opportunity Navigator
- Career Copilot
- Small Business Console
- Health Bureaucracy Copilot
- AI Action Ledger / Work Receipt

## Non-negotiable constraints
- Local-first default
- Low-resource mode (CPU/RAM friendly)
- No paid API dependency
- Human approval for external actions
- Cross-platform packaging target (**Windows, macOS, Linux**)
