# AI Toolkit ## What this project does
A practical enablement toolkit for teams adopting AI in operations workflows. It contains templates and structured assets for:
- process audits
- prompt library curation
- workflow standardization
- team onboarding artifacts ## Contents
- `01_Process_Audit_Template.docx` — audit framework for identifying automation opportunities
- `04_AI_Prompt_Library.docx` — curated prompt playbook
- `README.txt` — original internal notes ## Portfolio intent
This project is shown as a **process design and enablement artifact** rather than an executable codebase. It demonstrates governance and operationalization around AI usage. ## Repo cleanup notes
This public version removes secrets and runtime artifacts. ## Suggested improvements
- Convert docs to Markdown for easier review/version control
- Add examples with anonymized before/after outcomes
- Add a lightweight website/docs index for browsing
