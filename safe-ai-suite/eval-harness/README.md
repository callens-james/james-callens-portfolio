# Evaluation Harness

## Goal
Batch-test workflows and measure performance + reliability.

## MVP features
- Dataset of prompts/tasks with expected outcomes
- Run matrix across models/modes
- Metrics:
 - latency p50/p95
 - pass/fail rate
 - escalation rate
 - error taxonomy
- Save results to CSV/JSON for reporting

## Why this stands out
Proves engineering rigor and decision quality under constraints.
