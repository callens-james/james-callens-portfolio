# Case Studies Summary

## 1) Weather Dashboard (Desktop, Local-First)
- Problem: Practical daily planning for outdoor work/gardening without subscription dependency.
- Solution: Portable desktop app with multi-source weather aggregation, day scoring, and planning exports.
- Safety features: deterministic scoring, source health visibility, stale-data fallback.

## 2) R Workflow Suite
- Problem: Fragmented script-based workflows and weak repeatability.
- Solution: Organized R analysis suite with structured docs and cleaner project boundaries.
- Safety features: runtime artifact cleanup, reproducibility-focused structure.

## 3) Safe AI Stack (Tracks 1–4)
- Problem: Agent automation risk in production-like workflows.
- Solution: Risk Gate + Timeline Viewer + Eval Harness + Safe Automation Template.
- Safety features: policy gates, confirmation requirements, audit trails, measurable evaluation.
