Guided Onboarding (Start Here)

  1. Set your project path in First-Run Setup / Directory Picker, then click Initialize.
  2. Run System Check to confirm core services are healthy.
  3. Run Quick Self-Test from README commands to confirm warn/block behavior.
  4. Choose guard mode: keep shell trap ON (default) or toggle OFF with script.
  5. (Optional) Enable Telegram alerts via backend/.env.local.
