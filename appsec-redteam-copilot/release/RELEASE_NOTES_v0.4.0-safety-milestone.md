# Release Notes — v0.4.0 Safety Milestone

## Summary
This milestone shifts AppSec from advisory checks toward brokered execution governance with capability-scoped mutation control and verifiable audit integrity.

## Highlights
- Broker-only mutation path defaults
- Capability issue/inherit/validate model
- One-time command/context-bound approvals
- Workspace containment + protected-path floors
- Hash-chained audit log + integrity verify
- Out-of-band mutation monitor (alert-first)
- Safety metrics (broker coverage status)
- Emergency override lifecycle (explicit, time-boxed, audited)

## Notes
- AppSec governs integrated brokered mutation workflows.
- AppSec complements but does not replace host OS/EDR/sandbox controls.
